# Google Ads — Negative Keywords List

> Generated 2026-06-03 from the all-time Search Terms report (296 clicks, 2,840 impr, 2,370 MAD, 52 conv).
> Purpose: stop spending on searchers who will never hire a freelance developer (DIY/free/platform/foreign/competitor), lower CPA, and concentrate budget on buyers.
> **How to apply:** Google Ads → Campaigns → **Audiences, keywords and content** → **Negative keywords** → add to a shared list applied to all 3 campaigns (Casablanca / Rabat / Marrakech). Use the match type noted per section.

## ⚠️ Read first — do NOT negate these
